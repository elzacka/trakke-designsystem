## Bruk i kode

### SCSS

```scss
@use '@trakke/design-system/styles/colors' as *;

.my-element {
  background-color: $trk-brand;
  color: $trk-text;
  border: 1px solid $trk-border;
}
```

### CSS Custom Properties

```css
.my-element {
  background-color: var(--trk-brand);
  color: var(--trk-text);
  border: 1px solid var(--trk-border);
}
```

### JavaScript/TypeScript

```tsx
import { colors } from '@trakke/design-system/tokens';

const style = {
  backgroundColor: colors.brand,
  color: colors.text,
};
```

## Kontrastkrav

Alle fargekombinasjoner oppfyller WCAG 2.1 AA-krav for kontrast:

- Normal tekst: minimum 4.5:1
- Stor tekst: minimum 3:1
- UI-komponenter: minimum 3:1
